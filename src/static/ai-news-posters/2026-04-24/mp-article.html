01 · 营收融资
Anthropic 年化营收 300 亿美元，Claude Code 单破 25 亿
图 2 · 海报 01
一句话：估值破 1 万亿反超 OpenAI，Claude Code 成单兵增长引擎

发生了什么：Anthropic 年化营收从 2025 年底 90 亿冲到 3 月底 300 亿，Claude Code 单产品线年化 25 亿。

为什么重要：企业市场把 Claude 当编程团队的第二大脑，不是聊天玩具；营收结构比 OpenAI 更健康。

对你的影响：学 Claude Code 比学通用 Prompt 更值钱；面试官问 AI 工具，直接讲 Claude Code 实操。

来源：quantosei.com · businessinsider.com
02 · 模型竞争
Google DeepMind 组 Strike Team 追赶 Claude 编程
图 3 · 海报 02
一句话：Sergey Brin 亲自敲警钟：Claude 编程碾压 Gemini

发生了什么：Google 内部成立专项组改 Gemini 编程短板，Brin 发内部信承认 Claude 在代码任务上更强。

为什么重要：编程是 AI 模型 2026 年最能变现的场景，Google 不追上 Gemini API 的企业合同会继续流向 Anthropic。

对你的影响：选工具还是 Claude Code 优先；但盯 Gemini 下一个大版本，如果它真追上来价格战会来。

来源：sherwood.news · techcrunch.com
03 · 算力基建
Mira Murati 新公司签 Google Cloud 数十亿 GB300
图 4 · 海报 03
一句话：Thinking Machines Lab 锁 NVIDIA GB300 新芯片，押 Google 云

发生了什么：前 OpenAI CTO Mira Murati 的新 Lab 扩签 Google Cloud 合约，规模单位数十亿美元，用最新 GB300 GPU。

为什么重要：顶尖 AI Lab 离开 Azure 选 GCP 是路径信号——Google TPU + GB300 混合栈正成为前沿训练新标配。

对你的影响：学云选 GCP 的机会在变多；简历上 GCP Vertex AI / TPU 经验开始被企业直接点名要。

来源：techcrunch.com · bloomberg.com
04 · 模型发布
Kimi K2.6 开源，13 小时不间断编码 4000+ 行
图 5 · 海报 04
一句话：月之暗面最强代码模型，单次写 4000 行、全权重开源

发生了什么：月之暗面 4 月 20 日发布并开源 Kimi K2.6，测评中可连续 13 小时编码、生成或修改超过 4000 行代码。

为什么重要：国产开源模型第一次把 agent 级长程编码能力放到 Apache 风格许可下，私有部署的企业多了一个真能替代 Claude 的选项。

对你的影响：有 H100 / A100 的团队可以直接拉起来试；学生/独立开发者可以用 OpenRouter 上的免费额度先上手。

来源：21jingji.com · huggingface.co
05 · 营收融资
国内 AI 融资潮：面壁 10 亿、自变量 20 亿
图 6 · 海报 05
一句话：面壁智能跨独角兽门槛，具身智能赛道资本疯狂涌入

发生了什么：面壁智能一季度累计融资破 10 亿进独角兽；自变量机器人 B 轮近 20 亿，小米战投领投。

为什么重要：资本从「通用大模型烧钱」转向「能落地的端侧 + 具身智能」，2026 年国内 AI 投资主线换了。

对你的影响：想进国内 AI 公司，端侧模型 / 机器人 / 具身智能方向的岗位在急招；面通用 LLM 岗的竞争反而更卷。

来源：sina.com.cn · 21jingji.com
