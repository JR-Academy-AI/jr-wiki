## 1. AWS 扩展 AI 认证体系：Security Specialty 升至 SCS-C03，Generative AI Developer 正式开考

AWS 近期集中发布了两项影响在备考候选人的认证更新。

**SCS-C03：Security Specialty 的 AI 安全新考域**

现行 SCS-C02 的考点以传统安全服务为主——IAM 权限边界、VPC 网络隔离、KMS 密钥管理、GuardDuty 威胁检测。升级后的 SCS-C03 在这些基础之上，专门引入了针对 Amazon Bedrock、SageMaker 工作负载的安全治理模块：模型访问控制策略、AI 推理数据隔离、提示注入（Prompt Injection）防御，以及 AI 系统的审计日志架构设计。AWS 官方尚未公布 SCS-C02 的强制退役日期，但建议正在备考的候选人确认报名的是哪个版本，避免在切换窗口期踩坑。

**AWS Certified Generative AI Developer – Professional 标准考试开放**

该认证此前处于 Beta 阶段，现在标准考试正式开放报名。考试涵盖 Amazon Bedrock 模型选型与微调、RAG 架构设计、Agent 工具链编排（Function Calling / ReAct 模式）、负责任 AI 实践四个核心域。Professional 级别定价与 SAP-C03、DOP-C02 等同档，备考资料可参考 AWS Skill Builder 订阅中的官方增强准备包（Enhanced Exam Prep）。

## 2. Azure AB-620 AI Agent Builder Associate：微软首个 Copilot Studio 专项认证 4 月 Beta 上线

微软在 4 月 Partner Center 公告中正式确认 AB-620 AI Agent Builder Associate 进入 Beta 考试阶段，这是微软首个聚焦 **Copilot Studio 多 Agent 生产部署**的助理级认证。

**考试覆盖方向（Beta 版本预计范围）**

- Copilot Studio Agent 三种模式（Conversational、Autonomous、Multi-Agent）的选型与设计
- Microsoft 365 Copilot 扩展插件开发、知识源接入（SharePoint、Microsoft Graph API）
- Power Platform 集成：通过 Power Automate 编排 Agent 工具调用
- Entra ID 身份验证在 Agent 上下文的配置，以及 DLP 策略与 Agent 数据边界管控

**Beta 考试的实际价值**

Beta 版定价通常较正式版低约 25%（微软官方 Beta 折扣惯例）。考试结果需等待 2-3 周，但候选人可以在考后提交题目修改建议，影响正式版题库质量。对于 Power Platform 开发者和 M365 管理员，AB-620 是目前微软认证体系中覆盖 AI Agent 落地的最具针对性的考试。

**推荐先决条件**：已持有 PL-400（Power Platform Developer）或有 MS-700（Teams Administrator）基础的候选人，备考 AB-620 的曲线相对平缓，核心概念重叠度高。

来源：[Microsoft Learn – April 2026 Partner Center 公告](https://learn.microsoft.com/en-us/partner-center/announcements/2026-april)

## 3. Google Skills Arcade 4 月改版：积分刷课时代结束，云端资源验证成新标准

Google Skills Arcade 在 4 月发布平台规则重大变更，核心方向是**提高 Lab 任务的真实完成门槛**，解决此前大量用户机器刷积分的问题。

**主要变化**

任务完成判定机制从"页面停留时长 + 步骤截图"切换为"Google Cloud 控制台实际资源状态验证"——平台后台自动检测 Cloud 项目中是否真实创建了对应资源，截图操作不再被计入完成状态。Sprint 积分上限同步收紧，减少单日可累计的 Lab Credit 配额。

Gemini Enterprise Agent Ready（GEAR）计划的免费 Credit 不受影响：GEAR 订阅用户每月仍可获赠 35 个 Lab Credits，覆盖 Gemini API、Vertex AI Agent Builder 等认证备考核心路径。

**对备考 ACE / PCA 候选人的影响**

正在备考 Associate Cloud Engineer（ACE）或 Professional Cloud Architect（PCA）的候选人，这次改版实质上是利好——真实操作 Lab 的知识转化率远高于刷题积分，动手能力积累更扎实。Cloud Skills Boost 官方的结构化学习路径（Learning Paths）和认证模拟题（Practice Exams）功能均照常运行，不受此次改版影响。

来源：[Google Developer Community – Skills Arcade 4 月变更公告](https://discuss.google.dev/t/what-s-changing-in-the-google-skills-arcade-from-april-2026/345557) · [Google Cloud Skills Boost](https://www.cloudskillsboost.google/)

## 4. Cisco CCNP ENCOR 350-401 考纲升至 v2.0，CCIE Lab 引入 AI-ops 与零信任场景

Cisco 在 4 月完成了年度最大规模的职业级认证考纲刷新，影响覆盖 CCNP Enterprise、Security、Data Center 三条路径以及配套的 CCIE Lab 考试。

**CCNP Enterprise – ENCOR 350-401 v2.0 主要变动**

- **Network Assurance 模块**：新增 AI-driven Network Operations 场景，涉及 Cisco AI Network Analytics 与 ThousandEyes 的告警关联分析
- **Security 模块**：Zero Trust Architecture 从"了解"级别升至"配置与排错"级别，含 Cisco Duo MFA、ISE 集成配置场景
- **Virtualization 模块**：加入 Cisco Nexus Dashboard 多 Site 管理内容

**CCIE Enterprise Infrastructure Lab 新增场景**

Lab 考试（8 小时实机）引入两类新故障场景：AI-ops 告警驱动的自动化排错（要求配置 Cisco Catalyst Center 工作流）、SD-WAN 路由策略优化验证。对于已预约 Lab 的候选人，建议在考前确认所预约场次对应的是新版还是旧版考纲。

**切换窗口**：旧版 350-401 v1.x 题库的 Pearson VUE 下线日期暂未官方公布。正在备考的候选人可关注 Cisco Learning Network 官方通知，避免在新旧版本切换期间预约到错误的考纲版本。

来源：[Cisco Learning Network – A New Era for Cisco Certifications](https://learningnetwork.cisco.com/s/a-new-era-for-cisco-certifications)
