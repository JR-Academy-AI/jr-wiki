## 一、认证动态

### 1. SC-500 Beta 5 月 15 日开放 — AZ-500 八月退场，Azure 安全认证全面换代

**一句话**：微软把 AI 工作负载安全纳入核心安全认证，SC-500 Beta 本周五启动，AZ-500 还剩 4 个月有效期。

5 月 15 日，微软 SC-500（Microsoft Certified: Cloud and AI Security Engineer Associate）进入 Beta 阶段。这张认证是 AZ-500 的直接接班人，AZ-500 退休日期已定为 **2026 年 8 月 31 日**。

SC-500 和 AZ-500 最大的差异是考试范围：AZ-500 考的是云基础设施安全、身份与访问管理、网络安全和 SOC 运营这套传统体系；SC-500 在保留这些内容的基础上，新增了 **AI 工作负载安全**（AI Workload Security）、**AI 模型和 pipeline 的合规保护**（AI Governance）以及大语言模型部署场景下的安全架构设计。这个变化是有现实背景的：越来越多企业把 Azure OpenAI Service、Copilot Studio 接进了生产环境，但安全团队的技能覆盖还停留在几年前，SC-500 就是为这个缺口设计的。

Beta 阶段考试费按微软惯例约 USD $80（正式价 $165），适合已有 AZ-500 基础、想以较低成本提前拿证的学员。Beta 成绩完全有效，答案会在 GA 前校准。

两类学员的具体建议：现有 AZ-500 证书持有者，如果证书有效期在 2026 年底及以后到期，续证路径已关闭，建议直接开始备考 SC-500；正在备考 AZ-500 且进度超过 60% 的，8 月 31 日前考完，证书仍有效期 1 年；进度低于 40% 的，直接切换到 SC-500 备考资料，把 AI 安全相关模块提前补上。

### 2. AZ-204 七月三十一日退休 — Azure 开发者认证重心转向 AI，AI-200 接棒

**一句话**：AZ-204（Azure Developer Associate）7 月 31 日截止，替代考试 AI-200 把 Azure OpenAI 和 RAG 应用开发纳入核心考点，两者内容重叠约 60%。

微软确认 **AZ-204（Azure Developer Associate）退休日期为 2026 年 7 月 31 日**。接棒的考试代号是 **AI-200**，全名 Microsoft Certified: Azure AI Cloud Developer Associate。

从 AZ-204 到 AI-200 的变化本质上是微软对"Azure 开发者"角色的重新定位：AZ-204 覆盖 App Service、Container Apps、Azure Functions、Cosmos DB、Service Bus 等偏基础设施的开发技能；AI-200 保留了其中约 60% 的内容（容器化、API 集成、存储服务），同时把约 40% 的新内容压在 AI 开发场景上，具体包括：Azure OpenAI Service API 调用、Semantic Kernel 框架集成、Prompt Flow 构建 RAG 管道、Azure AI Studio 使用，以及 AI 应用的评估与监控。这个趋势和市场需求对上了——大多数公司的 Azure 开发岗已经开始要求具备 AI 集成能力，AZ-204 的考点已经跟不上。

AI-200 Beta 已在进行中，GA 版本预计和 AZ-204 退休节点相近，USD 报名费预计与 AZ-204 持平（$165）。

现实建议：AZ-204 备考进度超 60% 的，冲刺 7 月前考完，证书有效期 1 年；进度不到一半的，直接看 AI-200 备考材料，集中补 Azure OpenAI Service 集成和 AI Studio 操作这两块新内容；已有 AZ-204 证书的，目前仍然有效，续证考试会对应切换到 AI-200。

> 来源：[Microsoft Certification Exam Changes 2026 — mscertquiz.com](https://mscertquiz.com/blog/microsoft-certification-exam-changes-2026) · [Microsoft Certification Guide 2026 — lived365.com](https://www.lived365.com/post/microsoft-certification-guide-2026-newexams-retirements-aiskills-and-careerimpact)

### 3. AWS 考试 AUD 报价 5 月 15 日调整 — 手头有 voucher 的不受影响

**一句话**：AWS 5 月 15 日更新 AUD/EUR/CNY 汇率，美元价格不动，本周购入考试 voucher 可锁定当前澳洲价格。

AWS 官方宣布，**2026 年 5 月 15 日**起，通过 Pearson VUE 或 PSI 以 AUD、EUR、CNY 支付的 AWS 考试费将随当前汇率调整。美元定价维持不变：

| 级别 | USD 价格 | 涉及考试 |
|------|---------|---------|
| Foundational | $100 | CLF-C02、AIF-C01 |
| Associate | $150 | SAA-C03、DVA-C02、SOA-C02、MLA-C01 |
| Professional / Specialty | $300 | SAP-C02、DOP-C02、SCS-C03 等 |

以 Associate 级 USD $150 为基准，目前澳洲报价约 AUD $230–245 区间；调价后按近期汇率估算，AUD 价格会有小幅变动（具体金额取决于 5 月 15 日当日汇率）。对单次考试影响有限，但同期备考多门认证（比如同时推进 SAA-C03 + SCS-C03）的学员累计影响值得关注。

**关键保护**：已购买 voucher 的不受本次汇率调整影响，voucher 按购买时面值兑换。也就是说，**5 月 15 日之前购入的 voucher 锁定当前 AUD 价格**。购买渠道：Pearson VUE 账户直接注册考试并付款，或通过 AWS 授权培训合作伙伴购入 voucher。

### 4. AZ-802 Beta 开放 — 微软新增混合基础设施管理认证

**一句话**：AZ-802（Hybrid Server Administration）Beta 5 月开放，专门覆盖 Azure Arc + on-prem Windows Server 混合场景，7 月 GA。

微软 **AZ-802** 是 2026 年新增的基础设施方向认证，Beta 考试 5 月开放，正式 GA 和配套培训材料预计 7 月上线。

这张认证的定位填补了现有体系的一个缺口：AZ-104（Azure Administrator Associate）主要考纯云场景；而大量企业实际运行的是混合架构——部分工作负载在 Azure，部分仍在 on-prem Windows Server，用 Azure Arc 做统一纳管。AZ-802 专门覆盖这类混合场景，考点集中在：Windows Server 管理与 Azure Arc 集成、Hybrid Entra ID 跨环境身份、混合网络连接（VPN Gateway + ExpressRoute），以及 Azure Monitor 跨混合环境的统一监控与告警。

对于在金融、政府、医疗等行业工作、需要维护混合基础设施的运维工程师，AZ-802 比 AZ-104 更贴合实际工作内容。Beta 阶段考试费约 $80，GA 后预计 $165。

## 二、免费 / 限免课程

### 1. Hugging Face LLM Course — 大模型全链路，永久免费，无需注册

**适合**：有 Python 基础、想系统搞懂 LLM 从架构到部署全流程的学员
**链接**：[huggingface.co/learn/llm-course](https://huggingface.co/learn/llm-course/chapter1/1)

Hugging Face 把原来的 NLP Course 升级成了 LLM Course，使用 Hugging Face 生态的 Transformers、Datasets、Tokenizers 和 Accelerate 库作为动手工具。课程完全免费、无广告、不需要注册账号就能访问所有材料。

课程内容覆盖：Transformer 架构原理和 tokenization、用 SFT/LoRA/QLoRA 做模型 Fine-tuning、RLHF 和 DPO 对齐技术、LLM 评估与 benchmark 方法，以及 RAG 系统构建和向量数据库集成。每章约 6–8 小时，按官方建议每章 1 周完成，总计 5–6 周。所有代码示例跑在 Google Colab，不需要本地 GPU。

目前课程没有证书，但 Hugging Face 已确认正在开发认证程序，预计年内上线。有 AWS AIF-C01 或 Azure AI-900 基础证书在手、想真正进入技术层的学员，这门课是最直接的进阶路径，它不教你 AI 是什么，而是教你怎么改模型、怎么跑流程。

### 2. Google Cloud GEAR Program — 每月 35 积分，免费刷完 Skills Boost 全部实验室

**适合**：备考 GCP Associate Cloud Engineer 或 Professional Data Engineer 的学员
**链接**：[cloudskillsboost.google](https://www.cloudskillsboost.google/)

Google Cloud GEAR（Google Educational Access Resources）Program 每月提供 **35 免费学习积分**，可以用于访问 Google Skills 平台（原 Cloud Skills Boost）上几乎所有的实验室（Lab）和 Quest（系列学习路径）。

35 积分够覆盖一个月的主动备考量：标准 Lab 每个 2–3 积分，一个月可以完成 10–15 个；完整 Quest 约 10–15 积分，一个月可以打通 2–3 条路径。几条值得优先刷的路径：

- **Cloud Engineer Learning Path**（备考 Associate Cloud Engineer 主线，约 40 积分，两个月积分刷完）
- **Generative AI for Developers**（Vertex AI 和 Gemini API 实战动手，约 15 积分）
- **Applied ML: Building ML Pipeline on Google Cloud**（面向 Data Engineer，约 20 积分）

注册 GEAR Program 不需要信用卡，通过学术邮箱或提交申请表格即可。入口：cloudskillsboost.google → 右上角账户 → Subscription → GEAR Program。

### 3. AWS Skill Builder 免费路径 — 600+ 课，CLF-C02 到 Bedrock 应用全覆盖

**适合**：备考 AWS CLF-C02、AIF-C01，或想跟进 AWS 最新 AI 服务动态的学员
**链接**：[skillbuilder.aws](https://skillbuilder.aws/)

AWS Skill Builder 免费层（无需付费订阅）包含 600+ 课程。AI/ML 和备考方向的实用选项：

- **AWS Cloud Practitioner Essentials**：CLF-C02 备考标配，约 6 小时，覆盖所有考试域
- **AWS Certified AI Practitioner Study Path**：AIF-C01 官方备考路径，含概念讲解和场景练习
- **Introduction to Generative AI on AWS**：约 1 小时，快速建立 Bedrock / Amazon Q 概念框架
- **Amazon Bedrock Getting Started**：约 2 小时，含 API 动手实验，适合准备 Professional 级 AI 场景题的学员

结合本周的 AUD 调价节点，最合理的备考顺序是：先在 Skill Builder 免费课确认备考准备度，再在 5 月 15 日前购入 voucher 锁价。付费的 Skill Builder Individual 订阅（USD $29/月）额外开放考试模拟题和沙盒环境，短期冲刺阶段（考前 2–3 周）可以考虑订阅一个月，不必长期续费。
