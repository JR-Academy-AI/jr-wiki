## 1. Google 砸 400 亿美元投 Anthropic，估值锁 3800 亿

![Anthropic 与 Google 巨额投资](https://images.unsplash.com/photo-1677442136019-21780ecad995?w=1200)

**一句话**: 100 亿现金立即到账，剩下 300 亿与业绩里程碑挂钩，Anthropic 进入「Google + Amazon 双股东」时代。

Google 在 4 月 24 日正式公告将向 Anthropic 投资最高 400 亿美元 —— 100 亿现金立即注入，剩下 300 亿与未来若干个业绩里程碑达成情况挂钩。本轮对应 Anthropic 最新估值 3800 亿美元，比 2025 年底跳了一大截。Anthropic 同步披露年化营收已破 300 亿，Q1 单季环比再翻倍。

这笔交易真正值得读两遍的是它的结构信号。加上 Amazon 早前公告的 50 亿初期 + 最高 200 亿追加，Anthropic 现在同时拿着 Google 和 Amazon 两笔超大额承诺，累计超过 600 亿美元。这种「两个云大厂同时下注同一家 foundation model 公司」的结构，OpenAI 那边只有微软一家。换言之 Anthropic 已经走出了「靠 AWS 一家活着」的早期阶段。

对个人开发者，Anthropic 拿到这么多弹药短期会做三件事：模型迭代提速、企业产品扩张、订阅定价上调。学 [Claude Code 工作流](https://jiangren.com.au/learn/vibe-coding/hub) 和 prompt 资产沉淀的工程师红利窗口至少还有 12 个月，但建议同步把工具链做成 vendor-agnostic，别把所有鸡蛋放在 Anthropic 一家。

## 2. Anthropic 锁定 1000 亿美元 Trainium + 5GW TPU，10GW 双线绕开 Nvidia

![AI 算力数据中心](https://images.unsplash.com/photo-1518770660439-4636190af475?w=1200)

**一句话**: foundation model 公司第一次明牌「同时押 Trainium 和 TPU」，绕开 Nvidia GPU 主供给。

和 Google 那笔投资同一天宣布的，还有一份很容易被淹没但更值得读的算力大单。Anthropic 04-24 披露：与 Amazon 签订总额 1000 亿美元的算力承诺，将获得最高 5GW 规模的 AWS Trainium 训练 + 推理集群；与 Google 在本轮投资里同步拿到 5GW 的 TPU 容量。两家加起来 10GW —— 一座 1GW 的核电站可以给一座中等城市供电，Anthropic 现在锁定的是 10 座这种规模的电力级算力。

这个组合的战略意义远超数字本身。Anthropic 这次明牌「同时押 Trainium 和 TPU」 —— 等于把 Nvidia 排除在主供给之外。背后逻辑很清楚：训练侧 Nvidia H100/B200 仍然是天花板，但推理侧已经开始内卷，Trainium2 和 TPU v6 的每 token 推理成本已经比 H100 便宜 30-50%。Anthropic 押的是「未来 18 个月推理需求会是训练的 5 倍以上」，专用推理芯片的价格优势会被放大。

对工程师的实操线索：做 ML Infra / MLOps 的人，Trainium 工具链（AWS Neuron SDK）和 TPU 工具链（JAX-on-TPU、Pathways）的市场需求会跟上来 —— 这两条路线的工程师数量目前比 CUDA 程序员少一个数量级，上车窗口比当年学 CUDA 还宽。配合 [AI Engineer 学习路径](https://jiangren.com.au/learn/ai-engineer/hub) 把模型部署 + 多硬件 inference engine 这块技能补上。

## 3. Bezos AI 实验室融 100 亿美元，押注「物理世界 foundation model」

![机器人与具身智能](https://images.unsplash.com/photo-1485827404703-89b55fcc595e?w=1200)

**一句话**: 不做大语言模型，目标是训练能预测物理世界下一帧状态的 foundation model。

Financial Times 04-21 率先披露、Bloomberg 跟进确认：Jeff Bezos 个人主导的新 AI 实验室「Project Prometheus」即将完成一轮 100 亿美元融资。这家实验室不做大语言模型，目标方向是「能够理解物理世界的 foundation model」 —— 它要训练的不是「下一个 token 预测」，而是「下一帧物理状态预测」。Bezos 之前通过 Amazon 已经向 Anthropic 投了 200 亿美元，这次以个人身份再开 100 亿独立 AI lab，是把人生第三大笔资本押在 LLM 之外的下一个范式。

为什么 Bezos 不去拼通用对话模型？翻译成人话：通用对话模型这条赛道前面已经有 OpenAI、Anthropic、Google 三个亿级 ARR 玩家咬死，资本效率不如另开战场。具身智能、机器人控制、自动驾驶仿真，这三个方向恰好是 LLM 范式的盲区 —— 大模型很会写代码、写文案、解奥数，但让它操控机械臂叠衣服、让它仿真一辆车在雨天高速变道，效果都很差。Bezos 押的是「物理世界 foundation model」会成为机器人和自动驾驶的底层引擎，类似 GPT 之于聊天机器人。竞争对手不是 OpenAI，而是 FigureAI、1x、Skild AI、Physical Intelligence、Wayve 这一批。

对工程师：如果你在做机器人、自动驾驶、3D 视觉、物理仿真、SLAM、运动规划，2026 下半年到 2027 全年会迎来一轮大规模招聘 —— 这一波薪资中位数比纯 LLM 工程师高 30-50%。具体技能栈建议补：扩散模型用于视频/3D 生成、神经场 (NeRF / 3DGS)、强化学习用于机器人控制、Isaac Sim / Mujoco 物理仿真、ROS 2 + 多传感器融合。

> 来源: [Bloomberg](https://www.bloomberg.com/news/articles/2026-04-21/jeff-bezos-nears-10-billion-funding-round-for-ai-lab-ft-says)

## 4. OpenAI 推 ChatGPT 工作空间 Agent + GPT-5.5 定价翻倍

![OpenAI 工作空间 Agent](https://images.unsplash.com/photo-1526374965328-7f61d4dc18c5?w=1200)

**一句话**: ChatGPT 从「问答工具」升级成「数字员工」，正面对标 Google AI Agent 收件箱。

OpenAI 04-24 一口气发了两个消息。第一是 ChatGPT Business / Enterprise / Education 客户上线「工作空间 Agent」(Workspace Agents)：用户可以在 ChatGPT 里直接搭一个能跨 Slack、Gmail、内部知识库自动执行多步任务的 agent，agent 可以收集上下文、按工作流推进、在关键节点请求人工审批、产出可追溯的执行报告。第二是 GPT-5.5 公开定价：输入 $5 / 百万 token、输出 $30 / 百万 token，是 GPT-5.4 的整整 2 倍。

把这两件事放一起看，OpenAI 的产品路线图就清晰了。一年前 ChatGPT 还是「一个聪明的对话窗口」，现在已经在向「数字员工 + 数字团队」演进。这恰好和上周 Google Cloud 发布的「AI Agent 专用收件箱」是同一种产品形态：让 agent 有持久身份、有 inbox、有 audit trail，可以被多人多系统委派任务。两家正在企业 agent 这条线上正面开战。GPT-5.5 涨价则是另一条线 —— OpenAI 也在为可能的 IPO 做财务报表，「单 token 毛利」是最直接的杠杆。

对个人和团队的实操线索：纯靠 ChatGPT Plus 跑工作流的，提前预算上调 —— GPT-5.5 进 Plus 之后跑大任务的成本会明显涨，必要时做「Sonnet / GPT-4o 兜底 + 5.5 关键节点」的分级路由。prompt 工程时代正在结束，agent 设计时代开始 —— 你需要会写「多步骤工作流 + 工具调用 + 审批节点 + 失败回退」的 agent，光会调 API 已经不够。

## 5. 印第安纳大学 GenAI 101 全球免费开放，含证书 + AI 助教

![GenAI 101 在线课程](https://images.unsplash.com/photo-1503676260728-1c00da094a0b?w=1200)

**一句话**: 顶级研究型大学官方 AI 入门课，8 模块 / 16 课时，颁发证书，可写简历。

印第安纳大学 Kelley 商学院 04-16 宣布，把原本只面向校内学生和员工的 GenAI 101 课程，对全球公众完全免费开放。这门课从 2025 年 8 月在校内开课至今，已经累计招收超过 11.4 万学生。课程内容是 8 个模块 / 16 节课，覆盖 prompt engineering、数据叙事、AI 生成内容的事实核查、伦理使用、跨工作场景的 GenAI 应用。课程内置一个名叫 Crimson 的实时 AI 学习助教，学员可以随时问问题；通过课程后还颁发可写简历的证书。

为什么这门课比一般免费 AI 课值得读？三个原因。一，它由顶级研究型大学的商学院官方推出，写在 LinkedIn 和简历上立得住，HR 一查就能查到。二，课程内容刻意做成「不绑定单一厂商」 —— 同时教 Google Gemini 和 OpenAI ChatGPT。三，配套了一个对话式 AI 助教 Crimson，相当于把课程嵌进了一个支持问答的学习环境，比单纯看视频效果好得多。

实操建议：如果你是非技术背景但想系统学一下 GenAI，这门课是 2026 年低成本上车的最优解。学完后建议立刻配合 [JR Academy 的 Prompt Master 学习路径](https://jiangren.com.au/learn/prompt-master)，把 prompt 能力从「单次提问」升级到「prompt 工程 + agent 工作流」组合能力栈 —— 后者才是 2026 求职市场真正稀缺的技能。如果你已经有技术背景，建议跳过 GenAI 101 直接学 [AI Engineer 路径](https://jiangren.com.au/learn/ai-engineer/hub) + Anthropic / OpenAI 官方 cookbook。
