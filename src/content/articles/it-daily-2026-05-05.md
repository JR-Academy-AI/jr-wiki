## 一、认证动态

### 1. SnowPro Core COF-C02 五月十四日退场：还剩 9 天，立刻切换备考资料

**一句话**：今天是 5 月 5 日，COF-C02 英文考试在 9 天后（5 月 14 日）正式关闭，还在用旧题库备考等于在冲已停售的产品线。

Snowflake 于 2026 年 2 月 16 日正式上线 SnowPro Core 新版本 COF-C03，同时宣布 COF-C02 的退场时间表：**英文版 5 月 14 日截止，中日韩翻译版延至 7 月 31 日**。

新旧版本最明显的三个变化：

**1. 考试结构精简**：从 6 个域压缩为 5 个域，删掉了重复的考点，但整体题目数量维持在 100 题、115 分钟。

**2. 新增平台特性**：COF-C03 加入了 **Snowflake Notebooks**（原生 Python/SQL 混合开发环境，2025 年 GA）、**Apache Iceberg 表**（外部表格式支持，多云数据湖集成场景）两个考点，这两个功能在 COF-C02 里完全没有。

**3. AI 权重显著提升**：**Snowflake Cortex**（内置 LLM 推理和向量检索能力）在新版中有独立考点，包括 Cortex Analyst、VECTOR_COSINE_SIMILARITY 函数和 AI-assisted analytics 场景。这部分在 COF-C02 里只是边角题，COF-C03 把它提到了核心地位。

考试费用不变，$175 一次，无免费重考。Snowflake 官方 Snowflake University 的免费学习路径已同步更新到 COF-C03 版本。

行动建议：如果还在用 2025 年的 COF-C02 备考资料，立刻替换。Snowflake 官网 [learn.snowflake.com](https://learn.snowflake.com/en/certifications/snowpro-core-c03/) 有免费的 COF-C03 考试指南和学习路径，建议直接从官方 Exam Guide 对照新 5 个域逐一核查自己的盲点。

> 来源：[SnowPro Core COF-C03 — learn.snowflake.com](https://learn.snowflake.com/en/certifications/snowpro-core-c03/) · [SnowPro Core Quick Facts 2026 — certificationpractice.com](https://certificationpractice.com/exam-overviews/snowflake-snowpro-core-quick-facts)

### 2. AZ-800 + AZ-801 九月三十日退场：合并为单考 AZ-802，Beta 六月开放

**一句话**：两门考试变一门，已持证者只需续证评估，不用重考 AZ-802。

微软将把 **Windows Server Hybrid Administrator Associate** 认证的两门考试（AZ-800: Administering Windows Server Hybrid Core Infrastructure + AZ-801: Configuring Windows Server Hybrid Advanced Services）合并为单门考试 **AZ-802**，退场时间定为 **2026 年 9 月 30 日 17:00（美国中部时间）**。

AZ-802 Beta 预计 **6 月开放**，正式考试（Live）预计 **8 月上线**，报名入口届时会出现在 Microsoft Learn 认证页面。

**对不同人群的影响：**

- **正在备考 AZ-800 + AZ-801 的**：9 月 30 日前考完两门均有效，证书有效期 1 年。或者等 6 月 AZ-802 Beta 上线，一次考完，Beta 价约 $80（正式价 $165），成绩完全有效。
- **已拿到 Windows Server Hybrid Admin 证书的**：证书到期前只需完成 AZ-802 **续证评估**（Renewal Assessment），不是完整考试，在 Microsoft Learn 上免费完成，不收费。
- **刚开始规划的**：直接等 AZ-802，不必花时间备考注定要消失的两门旧考。

Windows Server 混合管理技能在澳洲企业客户场景里需求稳定，微软整合的动作不影响技能含金量，只是减少了考试成本。

> 来源：[AZ-801 vs AZ-802: Hybrid Server Admin Guide 2026 — certempire.com](https://certempire.com/az-801-vs-az-802/) · [Microsoft Certification Retirements 2026 — aicloudpartners.com](https://www.aicloudpartners.com/guides/microsoft-certification-retirements-2026.html)

### 3. AWS SAA-C03 Q2 2026 改版：弹性架构和 Bedrock 集成各占重头

**一句话**：SAA-C03 的场景题已深度植入 Bedrock，单纯靠记服务名字的备考策略在 2026 年的考场上会直接扣分。

AWS Solutions Architect – Associate（SAA-C03）在 Q2 2026 完成了本轮内容更新，主要变化集中在两个方向：

**弹性架构（Resilient Architecture）提至 30%**：这是四个考试域中权重最重的一个，新版场景题从多个架构方案里挑最优，评判标准同时考虑成本、容错能力和性能——你需要能在 "Auto Scaling + ALB vs ECS Fargate vs Lambda" 之间快速判断场景适配度，而不是背出每个服务的定义。

**Bedrock 和 Serverless AI 集成**：开发者域新增子域，专门考 **Amazon Bedrock 与 Lambda 的 Serverless AI 集成场景**。具体题型包括：给定一个生成式 AI 工作流，选择合适的 Bedrock Foundation Model 部署方式；RAG 系统里 S3 + Bedrock Knowledge Base 的架构选型；Lambda 触发 Bedrock 推理时的 IAM 权限设计。

备考资料建议：Tutorials Dojo 的 SAA-C03 Practice Exams 已更新到 2026 版（[portal.tutorialsdojo.com](https://portal.tutorialsdojo.com/courses/aws-certified-solutions-architect-associate-practice-exams/)），Adrian Cantrill 和 Stephane Maarek 的 Udemy 课程均已加入 Bedrock 和 Serverless AI 章节。AWS Skill Builder 上也有官方免费的 SAA 模拟题，直接反映新版考点分布。

与 May 3 提到的 Professional 级别更新（6 月 1 日起加入 Amazon Q 场景题）不同，SAA-C03 这次是 **Associate 级别**的改版，影响范围更广——全球每年 SAA-C03 考试量是 Pro 级别的 4-5 倍。

> 来源：[AWS Certified Solutions Architect 2026 Blueprint — aabiance.com](https://aabiance.com/aws-certified-solutions-architect-associate-exam-blueprint-2026-complete-guide/) · [AWS Certification Coming Soon — aws.amazon.com](https://aws.amazon.com/certification/coming-soon/)

### 4. Google Cloud 考试交付商从 Kryterion 切换至 Pearson VUE，题库将按 Next '26 更新

**一句话**：预约入口已变，Kryterion 的老链接不再有效；题库即将同步 Gemini Enterprise Agent Platform 等产品命名。

2026 年 2 月 26 日起，所有 Google Cloud 认证考试的预约和报名通过 **Pearson VUE** 完成（[pearsonvue.com](https://www.pearsonvue.com/us/en/microsoft/updates.html) 可查）。Kryterion（原 Webassessor）平台已关闭 Google Cloud 考试的新预约通道，3 月 2 日起只有 Pearson 渠道可用。

实操影响：
- **原来存的 Kryterion 书签或链接全部失效**，需要重新在 Pearson VUE 注册账号并绑定 Google 账户才能预约。
- Pearson 同时支持 **在线远程监考（OnVUE）和实体考场** 两种方式，澳洲主要城市均有 Pearson 考场，在线监考也已全面开放。
- 考试 Voucher 现在从 Pearson 直接购买。

题库方面，Google 宣布将基于 **Google Cloud Next '26** 公布的产品更新同步更新各认证的考点，涉及 **Gemini Enterprise Agent Platform** 新名称、**Google Cloud 数据与分析栈**重组后的服务命名。具体哪几门认证先更新、更新时间表尚未公布，但 Professional Machine Learning Engineer 的内容于 2026 年 4 月 2 日已经过审核确认为最新版。

备考建议：近期备考 GCP 认证（特别是 Professional Data Engineer、Professional ML Engineer、Cloud Digital Leader）的学员，以官方 [cloud.google.com/learn/certification](https://cloud.google.com/learn/certification) 上的最新 Exam Guide 为准，第三方平台更新速度不一。

> 来源：[Upcoming Change to Exam Delivery Provider — support.google.com](https://support.google.com/cloud-certification/answer/16803278?hl=en) · [Google Cloud Certifications in 2026 — netcomlearning.com](https://www.netcomlearning.com/blog/top-5-google-cloud-certifications)

### 5. Microsoft 九个新 AI 认证上线，七个旧证三月起陆续退场

**一句话**：微软在 3 月 6 日一次性宣布最大规模的认证改版，把 AI 技能嵌入所有技术角色，旧证持有者需要在退场前确认续证路径。

微软在 2026 年 3 月 6 日公布了新一轮认证大改版，核心动作是 **7 个旧认证退场、9 个新 AI 认证上线**，覆盖的技术角色包括：

- **Azure Databricks Data Engineer**（DP-750，与 Databricks 联合颁发，已在 5 月 3 日的日报里详细介绍，GA 中）
- **Cybersecurity Business Professional**（SC-730，面向非安全技术岗，首次将安全意识认证推至业务层）
- **MLOps Engineer**（新角色，Azure Machine Learning + GitHub Actions 全流程）
- **GenAIOps Engineer**（负责 Azure OpenAI Service 和 Prompt Flow 的生产运营）
- **SQL AI Developer**（SQL + Azure SQL MI + AI 数据服务组合，针对数据库开发者）
- **Azure Hybrid Infrastructure**（含 AZ-802，合并现有 AZ-800/AZ-801）

旧证退场的实操影响：持有 **MCSA、MCSE** 等 2019 年前的老证的学员不受影响（已永久有效）；受影响的主要是持有 AZ-500、AZ-204 等已宣布退场证书（SC-500 和 AI-200 分别是它们的接班人，上周日报有详细介绍）。所有退场认证均提供明确的续证路径，没有路径的直接废止，通过 [Microsoft Learn 认证退场页](https://learn.microsoft.com/en-us/credentials/certifications/retired-certifications) 可以查到每张证的具体退场日期和替代方案。

> 来源：[Microsoft's Next-Gen AI Certifications 2026 — whizlabs.com](https://www.whizlabs.com/blog/microsofts-next-generation-ai-certifications/) · [Microsoft Certification Guide 2026 — lived365.com](https://www.lived365.com/post/microsoft-certification-guide-2026-newexams-retirements-aiskills-and-careerimpact)

## 二、免费 / 限免课程

### 1. Databricks Academy 全线免费概览课程 + 学生专属权益

**适合**：正在备考 Databricks 认证、或想在简历上加数据工程 / GenAI 技能的学员
**链接**：[customer-academy.databricks.com](https://customer-academy.databricks.com/learn)

Databricks 官方在 Databricks Academy 开放了所有认证方向的 **Overview 课程（目录编号 DB005a）**，完全免费，注册账号即可访问，不需要信用卡：

- Databricks Certified Associate Developer for Apache Spark
- Data Analyst Associate
- Data Engineer Associate / Professional
- Machine Learning Practitioner
- Generative AI Engineer Associate

每门 Overview 课程时长约 3-5 小时，覆盖考试域分布、样题格式和核心知识点清单，相当于官方版考纲精讲。实际备考还需要配套 Databricks 文档和实战练习，但 Overview 课程足以让你在 3 小时内判断要不要考、需要补哪个模块。

**学生专属**：有大学邮箱的学员可以在 [databricks.com/resources/student-resources](https://www.databricks.com/resources/student-resources) 申请免费访问所有 **自学课程（Self-Paced Training）**，不只是 Overview，包括 Spark 编程、Delta Lake、MLflow、LLM Fine-tuning 等完整模块。申请通过后一般 1-2 个工作日开通。

## 三、JR Academy 学员行动建议

1. **SnowPro Core COF-C02 持有者**：5 月 14 日前不打算考的，立刻把备考材料切换到 COF-C03。Snowflake University 的官方学习路径（免费）已经更新，直接用那个做知识点对照，重点补 Snowflake Notebooks 和 Cortex AI 两块。

2. **AZ-800 / AZ-801 备考中途的**：两门考的进度超过 70% 的，9 月 30 日前考完；进度低于 50% 的，等 6 月 AZ-802 Beta 上线，一门 ~$80 搞定，省一门考试费和大量重复备考时间。

3. **AWS SAA-C03 准备中的**：Tutorials Dojo 的 Practice Exams 已更新，确认题目里有 Bedrock 和弹性架构场景题再下单；如果用的是 2025 年购买的旧套题，去对照 AWS 官方新 Exam Guide（aws.amazon.com/certification/certified-solutions-architect-associate）检查盲点。

4. **GCP 认证在排期的**：在 Pearson VUE（[pearsonvue.com](https://home.pearsonvue.com/google)）注册账号，旧 Kryterion 凭证不能用，提前绑定好 Google 账户，避免考前才发现预约入口变了。

5. **Databricks Academy 的学生**：用大学邮箱去申请免费的自学课程权限，Data Engineer Associate 这条路在澳洲就业市场认可度稳步上升，免费有 Databricks 官方材料打底，后续配合 JR Academy 的项目实战，认证含金量要高得多。
