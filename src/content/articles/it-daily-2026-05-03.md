## 一、认证动态

### 1. DP-750 正式发布：Microsoft × Databricks 首张联合认证 GA

**一句话**：Exam DP-750（Microsoft Certified: Azure Databricks Data Engineer Associate）Beta 结束，GA 版预计 5 月初开放全量报名，报名费 $165 USD，这是 Microsoft 和 Databricks 首次联合颁发的双品牌认证。

三月中旬，微软在官方 Skills Hub 博客宣布 Exam DP-750（Implementing Data Engineering Solutions Using Azure Databricks）开放 Beta。Beta 阶段前 300 名使用折扣码 `DP750Deltona` 报名费降至约 $33（80% off），Beta 窗口已于 4 月 2 日关闭。GA 版（正式版）预计 5 月初在 Microsoft Learn 开放报名，报名费 $165 USD（与 DP-700 同级），通过分线 700/1000。

**考试核心考点**：
- Delta Lake 表格架构与 Change Data Capture 增量加载
- Unity Catalog 数据治理与访问控制策略
- Databricks Asset Bundles 管道打包与部署
- Lakeflow Declarative Pipelines（声明式 ETL 管道）
- Azure Data Factory 与 Databricks 集成
- MLflow 实验追踪与模型注册表

**为什么值得考**：这张认证同时出现在微软 Azure Associate 证书列表和 Databricks 官方认证页面，通过后可获得两个徽章。对在澳洲就业市场上同时服务 Azure 和 Databricks 客户的数据工程师，这是性价比最高的"一考双认证"路径。JR Academy 数据工程课程学员的 Delta Lake 和 Unity Catalog 模块与 DP-750 考纲直接对应。

> 来源：[Microsoft Community Hub – New Azure Databricks Data Engineer Associate Certification](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-microsoft-certified-azure-databricks-data-engineer-associate-certification/4494114) · [Microsoft Learn DP-750 Study Guide](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/dp-750)

### 2. SC-730 进入 Beta：Security 终于有了面向普通员工的版本

**一句话**：Exam SC-730（Microsoft Certified: Cybersecurity Business Professional）5 月开放 Beta，7 月 GA，首次将安全认证定位于非 IT 岗位，法务、财务、HR 都是目标人群。

微软 3 月底宣布 SC-730，定位明确：不考防火墙配置，不考 SIEM 部署，专门考"处理敏感数据的普通员工在日常工作中如何做安全决策"。考试内容包括：识别社会工程攻击（钓鱼邮件、语音仿冒）、在 Microsoft 365 中安全共享文件和数据、Microsoft Purview 信息保护的实际操作、以及在使用 Microsoft Copilot 场景下的数据隐私合规判断。前置条件：无。不需要任何 IT 认证背景。

同一批进入 Beta 的还有 **AB-210（Dynamics 365 Sales AI Consultant Associate）**：专注 Microsoft Copilot for Sales 落地配置和 D365 Sales AI 自动化工作流，GA 预计 6 月。

Beta 阶段通常有约 80% 折扣，微软会在官方 Learn 页面发布折扣码，建议直接订阅 [Microsoft Learn Blog](https://techcommunity.microsoft.com/category/microsoft-learn) 获取通知。Beta 结束后需等 8–10 周才能收到成绩，但费用远低于 GA 版（$165）。

> 来源：[SC-730 Cybersecurity Business Professional – Vlad Talks Tech](https://vladtalkstech.com/microsoft-learning-and-credential-news/sc-730-microsoft-cybersecurity-certification-business-users/) · [New Microsoft Beta Exams 2026: AI-103 Now Live](https://mscertquiz.com/blog/new-microsoft-beta-exams-2026)

### 3. AWS Professional 双考六月升级：SAP-C02 和 DOP-C02 加入 Bedrock / Amazon Q 场景题

**一句话**：AWS 从 2026 年 6 月 1 日起更新 Solutions Architect – Professional（SAP-C02）和 DevOps Engineer – Professional（DOP-C02）考题，加入 Amazon Q 和 Bedrock 实际部署场景，距今不足 30 天。

具体变动：
- **SAP-C02**：新增"在现有企业架构中接入 Amazon Bedrock foundation model"和"Amazon Q for Business 集成多系统工作流"场景题，预计权重 10–15%。
- **DOP-C02**：新增 AI pipeline CI/CD 自动化场景，涉及 Bedrock 模型版本管理和 SageMaker Pipeline 触发器配置。
- 两场考试时长和总题数（75 题 / 180 分钟）不变。
- 6 月 1 日前报考并完成考试的候选人不受影响；6 月 1 日及之后首次考试的必须覆盖新内容。

AWS Skill Builder 月订阅（$29/月）内的官方 Exam Prep Lab 已更新部分题目，建议本周登录对照 [Exam Guide PDF](https://aws.amazon.com/certification/certified-solutions-architect-professional/)（官方页面免费下载）核查新域权重。2025 年出版的备考书和 dump 网站的题库无法覆盖这块，用旧题库备考风险较大。

> 来源：[AWS Certification Coming Soon](https://aws.amazon.com/certification/coming-soon/) · [AWS revamps 2026 certification path with AI-focused exams – MSN](https://www.msn.com/en-us/news/other/aws-revamps-2026-certification-path-with-ai-focused-exams/gm-GM1D1C513F)

### 4. CKA 切至 Kubernetes v1.35：Gateway API 和 etcd 快照恢复是新增必考点

**一句话**：Linux Foundation CKA 考试环境已更新至 Kubernetes v1.35（2026 年 3 月起），新增 Gateway API 实操配置和 etcd backup/restore 考点，2024 年及以前的备考视频内容存在盲区。

**关键变化**：
- Kubernetes 版本：v1.34 → **v1.35**
- 新增考点：Gateway API（Ingress 的继任者，HTTPRoute / GatewayClass 配置）、etcd 快照创建与恢复操作（`etcdctl snapshot save / restore`）、kubeadm 升级流程（control plane 先于 worker node）
- 考试格式不变：17–25 道 performance-based 实操题，2 小时，通过线 66%
- 报名费：**$445 USD**，含一次重考（retake）机会
- 考试平台：PSI 在线监考，不支持 Pearson VUE

killer.sh 模拟环境（报名后赠送 2 次，每次 36 小时访问权）已同步更新至 v1.35，是目前最接近真实考试的练习平台。不推荐用任何 2024 年前的教程，直接用 [killer.sh 的官方课程材料](https://killer.sh/cka) + Linux Foundation 官方学习路径。

> 来源：[CKA Program Changes – Linux Foundation](https://training.linuxfoundation.org/certified-kubernetes-administrator-cka-program-changes/) · [CKA Exam Report 2026 – DEV Community](https://dev.to/suzuki0430/cka-certified-kubernetes-administrator-exam-report-2026-dont-rely-on-old-guides-mastering-the-534m)
