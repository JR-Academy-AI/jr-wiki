## 1. AWS Certified Generative AI Developer - Professional 标准考试刷新，正式纳入 Bedrock AgentCore

AWS Certified Generative AI Developer - Professional 原先以 Beta 形式运行，覆盖 Amazon Bedrock 基础模型调用、Prompt Engineering 和 RAG 架构设计。随着 Amazon Bedrock AgentCore 正式 GA，AWS 已将 AgentCore 纳入标准考试考纲，标准版现已开放报名。

**考纲变化关键点：**

- 新增 Amazon Bedrock AgentCore 架构与 Agent 编排考点
- 覆盖 AI 治理（AI Governance）框架与合规场景
- 标准考试取代 Beta，题库与当前 AWS AI 服务栈同步

**备考建议：**

AWS Skill Builder 上的官方备考课程（标准版免费，增强版需 $29/月订阅）已同步更新。若已购买 Beta 阶段备考资料，重点补充 AgentCore 和 AI 治理部分。Pearson VUE 和 PSI 均可报考，考前在 AWS 官网确认最新题纲版本。

来源：[AWS Certification — Coming Soon](https://aws.amazon.com/certification/coming-soon/) · [AWS Training and Certification Blog（Announcements）](https://aws.amazon.com/blogs/training-and-certification/category/post-types/announcements/)

## 2. AZ-305 考纲 2026-04-17 刷新，Azure Solutions Architect Expert 与 AI-901 同步更新

Microsoft 于 2026-04-17 完成 AZ-305 英文考纲更新，Azure Solutions Architect Expert 认证随之升级。同期，AI-901（Azure AI Fundamentals）Beta 考试也于 2026-04-15 完成刷新，两份更新均强化了生成式 AI 在 Azure 架构中的应用场景。

**备考影响：**

- 2026-04-17 之后安排 AZ-305 考试的候选人，将面对更新后的题库
- 备考资料购于 3 个月前的，建议重新核对 Microsoft Learn 官方学习路径的变更说明
- AI-901 Beta 是面向 AI 基础认证的入门考试，对准备 Azure AI Engineer（AI-102）的候选人具有参考价值

**学习资源：**

Microsoft Learn 官方学习路径已免费更新，模拟练习题优先使用 Microsoft Learn 官方练习测试或 MeasureUp，避免使用第三方题库（容易与旧版本不匹配）。

来源：[Azure Solutions Architect Expert — Microsoft Learn](https://learn.microsoft.com/en-us/credentials/certifications/azure-solutions-architect/) · [AI-901 Beta — Microsoft Learn](https://learn.microsoft.com/en-us/credentials/certifications/exams/ai-901/)

## 3. Azure Developer Associate（AZ-204）宣布 2026-07-31 退役，持证者须提前续期

Microsoft 官方确认 Azure Developer Associate 认证及对应考试 AZ-204 将于 **2026 年 7 月 31 日**正式退役。退役后，新报名入口关闭，已持有认证者无法通过续期保留该资质，认证状态将变为"已退役"。

**受影响人群操作指引：**

| 情况 | 建议行动 |
|------|---------|
| 尚未持有 AZ-204，计划报考 | 在 2026-07-31 前完成报考并通过 |
| 已持有，认证将在 7 月前到期 | 通过 Microsoft Learn 免费在线评估完成续期 |
| 已持有，认证到期日在 7 月后 | 退役后续期通道关闭，认证届时自动到期无法续期 |

**后续认证路径：**

Azure Developer 相关技能的接替考试尚未官方公布。建议持续关注 Microsoft Learn 认证公告页，以及 2026 年 February 新增的四项 Microsoft Certifications 是否涵盖替代方向。

来源：[Azure Developer Associate — Microsoft Learn](https://learn.microsoft.com/en-us/credentials/certifications/azure-developer/)

## 4. Cisco CCNP Enterprise 与 CCIE Lab 2026 更新，AI 网络分析与 Crosswork 自动化进入考场

Cisco 延续"新一代认证"计划，CCNP Enterprise 核心考试（350-401 ENCOR）及多项专向考题纲已于 2026 年更新，CCIE Enterprise Infrastructure 与 CCIE Security Lab 同步引入新的实战任务模块。

**CCNP Enterprise 更新重点：**

- **350-401 ENCOR**：新增 Cisco Catalyst Center 自动化工作流、AI 辅助网络分析场景覆盖
- **CCNP Security SCOR**：补充 Cisco Security Cloud 集成与机器学习威胁检测模块
- 增加 IPv6 企业网络部署深度考核

**CCIE Lab 更新重点：**

- **CCIE Enterprise Infrastructure Lab**：新增 Cisco Crosswork Network Automation 实战任务
- **CCIE Security Lab**：增加 Cisco Secure Firewall Threat Defense Virtual 部署实操
- Lab 整体更强调云原生网络和基础设施自动化场景

**备考建议：**

备考前以 Cisco Learning Network 官方路线图为准，及时核对最新考纲版本号。Cisco U.（原 dCloud）提供免费 30 天试用，可用于 Crosswork 和 Catalyst Center 上机练习，是 Lab 备考的重要免费资源。

来源：[Cisco Certification Roadmaps](https://learningnetwork.cisco.com/s/cisco-certification-roadmaps) · [A New Era for Cisco Certifications](https://learningnetwork.cisco.com/s/a-new-era-for-cisco-certifications)
