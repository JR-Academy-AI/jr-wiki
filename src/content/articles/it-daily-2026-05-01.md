## 1. AWS Certified Generative AI Developer – Professional 标准版开放报名

AWS 于 2026 年 3 月结束 Beta 测试，正式开放 AWS Certified Generative AI Developer – Professional 标准版报名。这是 AWS 认证体系中第一张以生成式 AI 工程实践为核心的 Professional 级证书。

考试覆盖四个主要域：Foundation model 选型与 Amazon Bedrock 微调、RAG（检索增强生成）管道设计、AI 推理性能优化、负责任 AI 部署（模型评估与数据合规）。题量 65 道，考试时长 170 分钟，报名费 $300 USD，支持 Pearson VUE 和 PSI 两家考场（含在线监考）。

AWS 官方推荐路径：持有 Developer – Associate 或 Machine Learning – Specialty 后再报考，实际工作经验至少 2 年。备考材料方面，AWS Skill Builder 月订阅（$29/月）包含官方 Exam Prep Lab 和模拟题集；免费渠道可下载官方考试指南（Exam Guide PDF）核对域权重分布。

对于正在用 Amazon Bedrock 构建 RAG chatbot 或 Agent 应用的工程师，这是目前 AWS 体系里最直接对应实际交付能力的认证路径。

来源：[AWS Certification – Coming Soon](https://aws.amazon.com/certification/coming-soon/) · [AWS Training Blog – March 2026 New Offerings](https://aws.amazon.com/blogs/training-and-certification/march-2026-new-offerings/)

## 2. Azure Developer Associate（AZ-204）7 月 31 日正式退役

Microsoft 官方确认：Azure Developer Associate 认证及 AZ-204 考试将于 **2026 年 7 月 31 日**退役。退役后新考生无法报名，现有持证者的更新评估（Renewal Assessment）同步下线，证书到期后不可续期。

受影响三类人：第一，备考中的考生，建议在 7 月底前完成考试，否则对应的已下线考纲版本内容备而无用；第二，证书到期时间在 2026 年 8 月及以后的持证者，更新通道关闭，证书最终会过期，Microsoft 目前未公布直接替代路径；第三，企业内部采购了 AZ-204 培训合同的技术团队，需核查交付时间节点。

Microsoft 尚未宣布替换 AZ-204 的新 Developer 专向证书。Azure 开发方向的候选人可参考的替代路径：Azure Solutions Architect Expert（AZ-305）覆盖架构决策层面，或通过具体场景的 Applied Skills 凭证（如 Build distributed apps with Azure Cosmos DB）积累技能证明。

从现在到 7 月 31 日只剩约 3 个月，已列入今年目标的考生需立即排期。

来源：[Microsoft Learn – Azure Developer Associate](https://learn.microsoft.com/en-us/credentials/certifications/azure-developer/)

## 3. AZ-305 英文版 4 月 17 日换新 + Microsoft 发布两张 AI 入门证书

**AZ-305 考纲更新**：Azure Solutions Architect Expert 核心考试 AZ-305 英文版于 2026 年 4 月 17 日完成最新一轮考纲调整，加入了 Azure OpenAI Service 集成设计场景，以及混合云 AI 工作负载的数据主权合规要求。非英语版本将分批跟进。正在备考 AZ-305 的考生建议重新核对官方 Exam Topics 页面，确认新增内容权重。

**两张新 AI 认证已正式 GA**：Microsoft 在 2026 年 2 月发布，目前均已正式上线：

- **AB-900 · Microsoft 365 Certified: Copilot and Agent Administration Fundamentals**：入门级，面向 IT 管理员，验证管理和保护 Microsoft 365 Copilot 及 AI Agent 服务的运维能力。
- **AB-730 · Microsoft Certified: AI Business Professional**：面向业务用户，验证用 Generative AI 和 Copilot 优化工作流程的能力，无需编程背景。

两张证书均通过 Pearson VUE 报名，考试费与 Azure Fundamentals 系列一致（$165 USD）。对没有深度技术背景、但需要向客户或管理层展示 AI 能力的从业者，AB-730 是成本最低的 Microsoft AI 认证入口。

来源：[Microsoft Community Hub – Credentials Roundup Feb 2026](https://techcommunity.microsoft.com/blog/skills-hub-blog/microsoft-credentials-roundup-february-2026-edition/3666867) · [Microsoft Learn – Azure Fundamentals](https://learn.microsoft.com/en-us/credentials/certifications/azure-fundamentals/)

## 4. Cisco 2026 全线改版：ENCOR 升 v2.0 · DevNet 更名 Automation 系列

Cisco 在 2026 年 4 月第二周正式推出 CCNP Enterprise 核心考试 350-401 ENCOR v2.0，配合 CCNP Security 各 concentration 考试同步调整（加入 zero-trust architecture 目标域），标志着 Cisco 2026 年认证体系最大规模的一次整体换新。

**350-401 ENCOR v2.0**：较上一版本强化了 Catalyst Center（原 DNA Center）智能运维场景和 SD-Access 相关内容，部分已过时的 DMVPN 传统拓扑题减少。官方 exam topic 已同步更新，备考材料中第三方出版书籍若为旧版（v1.x 考纲）需交叉核对。

**DevNet 更名**：自 2026 年 2 月 3 日起，Cisco DevNet 系列认证正式更名：DevNet Associate → CCNA Automation，DevNet Professional → CCNP Automation，DevNet Expert → CCIE Automation。课程结构不变，只是品牌对齐到 Cisco 主认证体系。已报名 DevNet 考试的考生，Pearson VUE 系统中证书名称会反映新命名，不影响考试内容。

**CCNP / CCIE Wireless**：2026 年 3 月 19 日起，CCNP Wireless 和 CCIE Wireless 资格重新开放注册，有无线网络方向需求的工程师可参阅 Cisco Learning Network 确认报名条件。
