## 定价和计划

Bolt.new 的定价体系（2026 年）：

| 计划 | 月费 | Token 额度 | 核心权益 |
|------|------|-----------|---------|
| **Free** | $0 | 30 万/天，100 万/月 | 基础功能，公开项目 |
| **Pro** | $25/月 | 1000 万/月 | 私有项目，优先速度 |
| **Teams** | $30/人/月 | 1000 万+/人 | 团队协作，共享模板 |
| **Enterprise** | 定制 | 定制 | SSO、审计日志、SLA、专属支持 |

**Token 滚存规则**（2025 年 7 月起）：付费计划的未用完 token 可以滚存到下个月，最多保留两个月有效。免费计划的 token 不滚存，每日额度每天重置，月度额度每月 1 号重置。

## Token 到底是什么

Token 是 AI 处理文本的最小单位。一个中文字大约 1-2 个 token，一个英文单词大约 1 个 token。

Bolt.new 里消耗 token 的动作：

- **发送 prompt**：你写的文字本身消耗 token
- **AI 生成代码**：生成的每一行代码都消耗 token
- **AI 读代码**：每次交互时，AI 要读一遍项目里的相关文件

这就是为什么项目越大越费 token——不是你写的 prompt 变长了，而是 AI 每次都要"重新理解"整个项目。

```
# 粗略估算
新项目第一条 prompt：~10,000 tokens
小项目（5个组件）：每次 ~30,000 tokens
中型项目（20个组件）：每次 ~100,000 tokens
大项目（50+组件）：每次 ~200,000+ tokens

# 免费计划 100 万月度 token ≈ 小项目 33 次交互
# Pro 计划 1000 万月度 token ≈ 小项目 333 次交互
```

## 常见问题

**Q：Token 用完了怎么办？**

A：免费计划的每日 token 第二天自动重置。月度 token 下个月 1 号重置。如果急着用，升级到 Pro 立即获得 1000 万 token。没有"单独购买 token"的选项，只能升级计划。

---

**Q：Bolt.new 生成的代码质量怎么样？**

A：说实话，中等偏下。跟 Cursor 这种面向专业开发者的工具比，Bolt.new 生成的代码会有更多冗余、命名不太规范、组件拆分不够合理。但对于 MVP 和原型来说完全够用。

如果你打算把 Bolt.new 生成的代码长期维护，建议：
1. 生成初版后下载到本地
2. 用 Cursor 或 VS Code 做一轮代码重构
3. 后续的精细开发在本地进行

---

**Q：项目越来越大，AI 开始出现奇怪的修改怎么办？**

A：这是所有 AI 编辑器的通病——上下文窗口有限，项目太大 AI 就"记不住"全貌。应对方法：

1. 一个 Bolt.new 项目尽量不超过 30-40 个文件
2. 让 AI 生成 README.md 记录项目结构
3. 每次 prompt 里指明要改的文件和组件名
4. 大项目考虑拆成多个小项目

---

**Q：Bolt.new 生成的代码有版权问题吗？**

A：根据 Bolt.new 的服务条款，你完全拥有生成代码的所有权，可以商用、分发、修改。

---

**Q：能用手机/iPad 操作 Bolt.new 吗？**

A：技术上可以打开网页，但体验很差。WebContainers 需要比较强的计算资源，在移动设备上会很慢。推荐用电脑操作。

## Bolt.new 和 Lovable 怎么选

这两个工具经常被拿来比较。我的看法：

| 你的情况 | 选 Bolt.new | 选 Lovable |
|---------|------------|-----------|
| 想用 Vue/Svelte | ✅ | ❌（只有 React） |
| 零代码经验 | 可以 | ✅ 更友好 |
| 在意 UI 质量 | 一般 | ✅ 更精致 |
| 预算有限 | ✅ $25/月 | $39/月 |
| 想理解代码 | ✅ 编辑器更透明 | 代码区不突出 |
| 需要 Supabase | 两者都支持 | 两者都支持 |

一句话总结：**Lovable 像设计师做的产品，Bolt.new 像工程师做的产品**。Lovable 出来的东西更好看，Bolt.new 给你的控制权更多。

## Bolt.new + Cursor 组合用法

这两个工具不是竞争关系，放在一起用效果很好：

```bash
# 1. 在 Bolt.new 里快速生成初版（1-2 小时）
#    用对话迭代出核心功能和基础 UI

# 2. 下载代码到本地
#    Bolt.new 项目设置 → Download

# 3. 用 Cursor 打开项目做精细开发
cd your-bolt-project
cursor .

# 4. Cursor 里做的事：
#    - 重构 AI 生成的冗余代码
#    - 添加复杂业务逻辑
#    - 写测试
#    - 性能优化
#    - 配置 CI/CD
```

Bolt.new 负责"从 0 到 1"，Cursor 负责"从 1 到 100"。
