## 定价和计划（2026 年 4 月）

| 计划 | 月费 | 含 Credits | 协作人数 | 核心权益 |
|------|------|-----------|---------|---------|
| **Starter** | $0 | 每日有限额度 | — | 10 个开发项目，1 个发布项目，代码公开 |
| **Core** | $25/月（年付 $20/月） | $25/月 | 5 人 | 全功能 Agent，私有项目，自定义域名 |
| **Pro** | $100/月（年付 $95/月） | $100/月 | 15 人 + 50 查看者 | Credit 滚存，优先支持，28 天数据库恢复 |
| **Enterprise** | 询价 | 定制 | 定制 | SSO/SAML，最高 64 vCPU / 128GB 内存 |

**学生折扣**：用 `.edu` 邮箱认证后，Core 计划 $10/月，优惠 6 个月。教育工作者（K-12 公立学校老师）免费，其他教育者 80% 折扣。

## Credits 到底怎么算

Replit 在 2026 年初从"每个 Checkpoint $0.25"改成了 **Effort-Based Pricing**（按工作量计费）。简单说就是：任务越复杂，消耗的 credits 越多。

```
# 粗略参考（实际费用因项目而异）

改个按钮颜色（Lite 模式）：~$0.06
标准功能开发（Economy）：~$0.25
复杂功能集成（Power）：$1-5+
大型架构任务（Turbo）：$5-30+

# 费用 = AI 模型调用 + 计算资源 + 出站流量
# Extended Thinking 和高功率模型会额外消耗 2-5 倍
```

**Credits 用完了怎么办**：可以购买额外的 Credit Pack，或者按量付费（超出部分自动计费）。注意：Core 计划的 credits 不滚存，月底清零；Pro 计划可以滚存一个月。

## 部署费用（单独计算）

部署费和 AI credits 是分开收的：

| 类型 | 起步价 | 适合 |
|------|--------|------|
| **Autoscale** | ~$1/月 | 流量不固定的 Web 应用（没人访问时不收费） |
| **Reserved VM** | ~$20/月 | 需要 24/7 在线的服务 |
| **Static** | 极低 | 纯前端站点 |

低流量应用大概 $5-10/月，高流量可能 $50+/月。

## 常见问题

**Q：免费计划够用吗？**

A：够体验，不够正经用。Starter 计划大约给你 10 个免费 Checkpoint 和每天有限的 AI 额度，足够做一个简单的 demo 感受一下。但如果你想持续开发，Core 是最低门槛。学生用 $10/月的折扣价很划算。

---

**Q：Replit Agent 生成的代码质量怎么样？**

A：对 MVP 来说够用，但别指望生产级别。常见问题包括：缺少错误处理、性能优化不足、安全加固不到位。如果你打算长期维护，建议：

```bash
# 推荐工作流：Replit 起步 → 本地精修

# 1. 用 Replit Agent 快速搭出 MVP
# 2. 下载代码到本地
git clone your-replit-project

# 3. 用 Cursor 或 VS Code 做代码审查和重构
cursor .

# 4. 后续精细开发在本地进行
# 5. 部署用自己的基础设施（AWS/Vercel/Fly.io）
```

---

**Q：Agent 说它改了代码但实际没改怎么办？**

A：这是已知问题，叫"Phantom Changes"——Agent 声称做了修改但文件实际上没变。解决方法：直接在文件管理器里点开对应文件检查，不要只看 Agent 的聊天回复。

---

**Q：项目越来越大，Agent 开始出错怎么办？**

A：所有 AI 编程工具都有上下文窗口的限制。项目超过 15-20 个组件后，Agent 就容易"记不住"全貌。建议：
1. 项目文件数控制在 30-40 个以内
2. 让 Agent 生成 README 记录项目结构
3. 每次 prompt 指明具体要改的文件和函数名
4. 用 Agent Skills 记录项目的技术规范和约定

---

**Q：Replit 宕机了 Agent 还能用吗？**

A：Agent 底层依赖 Anthropic 的 Claude API。如果 Claude 宕机（2025 年 Q4 发生过两次），Agent 会降级到 Gemini 模型，质量会明显下降。代码不会丢（保存在 Replit 的文件系统里），但建议重要项目定期下载备份。

## 和 Cursor 怎么选

| 你的情况 | 选 Replit Agent | 选 Cursor |
|---------|----------------|-----------|
| 零代码基础 | ✅ Agent 自主程度高 | 有门槛 |
| 想让 AI 全自动 | ✅ 描述需求就行 | 需要你参与决策 |
| 已有大型代码库 | 不合适 | ✅ 专业 IDE，完美支持 |
| 在意代码质量 | 一般 | ✅ 生成的代码更规范 |
| 不想管部署 | ✅ 一键部署，自带托管 | 部署自己搞 |
| 总成本（含托管） | $25/月全包 | $20/月 + 托管费另算 |

一句话：**Replit Agent 是"我有个想法，帮我做出来"；Cursor 是"我在写代码，帮我写得更好"**。

## 和 Bolt.new 怎么选

| 你的情况 | 选 Replit Agent | 选 Bolt.new |
|---------|----------------|-------------|
| 要做 Python/Java/Go 项目 | ✅ 50+ 语言 | 只支持 Web |
| 要做移动端 App | ✅ 支持 | 不支持 |
| 只要最快出个 Web 页面 | 稍慢 | ✅ 脚手架速度最快 |
| 需要数据库 | ✅ PostgreSQL 内置 | 要自己接 Supabase |
| 后续要持续迭代 | ✅ 完整 IDE 环境 | 复杂项目容易出问题 |
| 想用 Vue/Svelte | Replit 也支持 | ✅ 框架自由度高 |
