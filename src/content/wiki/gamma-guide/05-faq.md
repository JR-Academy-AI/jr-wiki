## 套餐对比（2026 年 4 月最新）

Gamma 的定价分个人版和团队版两条线：

```
个人版：

Free    → $0      | 400 积分（一次性）| 带 Gamma 水印 | 基础 AI 模型 | 不能导出 PPTX
Plus    → $10/月  | 1000 积分/月    | 去水印        | 高级图片模型 | 可导出 PPTX/PDF
Pro     → $20/月  | 4000 积分/月    | 自定义品牌     | API 访问 + 分析面板
Ultra   → $100/月 | 20000 积分/月   | AI 动画        | 75 张卡片/单文档 | 抢先体验新功能

团队版：

Team     → $20/人/月（最少 2 人）   | 共享品牌主题 + 文件夹
Business → $40/人/月（最少 10 人） | SSO + 数据不用于 AI 训练 + SOC 2
Enterprise → 联系销售              | 全定制
```

**年付打八折**，比如 Pro 年付是 $15/月。

## 最常见的 10 个问题

### 1. 免费的 400 积分够干什么？

大约能生成 10 份完整演示文稿。每次 AI 生成（包括生成新内容、AI 换图、Agent 重写）消耗积分。手动编辑文字、拖拽排版、换颜色这些不消耗积分。

**省积分的方法**：用 Paste 模式（自己写好内容让 AI 排版）比 Generate 模式（让 AI 从零生成）省；生成前把大纲调到位，避免反复重新生成。

### 2. 导出 PPTX 效果怎么样？

说实话，不太行。具体问题：

| 会丢失的 | 不会丢的 |
|---------|---------|
| Toggle 折叠区域 | 基本文字和图片 |
| Nested Card 嵌套卡片 | 简单表格 |
| 嵌入的视频 / Figma | 基本图表 |
| 复杂自定义字体 | 标准布局 |
| AI 动画效果 | 颜色和基本样式 |

**我的建议**：如果对方能用浏览器，发链接。如果一定要文件，导出 PDF 比 PPTX 靠谱。如果必须用 PPTX，生成时就选简单的主题模板，别用太花哨的排版。

### 3. 中文内容生成质量如何？

Gamma 在 2025 年底加了中文排版优化（宋体 / 微软雅黑字体支持、中文行距和字间距调整、中文语义理解优化），所以现在中文内容的生成质量比早期好了很多。

但是有几个地方要注意：
- **英文质量仍然更好**——如果你做的是双语内容，建议先用英文生成再让 Agent 翻译
- **专业术语有时候翻译很奇怪**——技术内容建议在 prompt 里直接写中英混合（"生成一份关于 Kubernetes 集群管理的演示"比"生成一份关于容器编排的演示"效果好）
- **长文本卡片排版**——中文因为没有空格分词，偶尔会出现换行不自然的问题，手动调一下就好

### 4. Gamma 的内容会被用来训练 AI 吗？

- **Free / Plus / Pro / Ultra（个人版）**：会被用于改善 Gamma 的 AI 模型
- **Business / Enterprise（团队版）**：明确承诺不会用于训练
- Gamma 已通过 SOC 2 Type II 认证，数据传输加密，符合 GDPR / CCPA

如果你的内容涉及公司机密，要么用 Business 套餐，要么别在 Gamma 上放敏感信息。

### 5. 能不能离线用？

不能。Gamma 完全是在线工具，没有桌面应用，没有离线模式。编辑和演示都需要联网。如果你经常在飞机或没网的地方工作，这是个硬伤。

### 6. 一份演示最多能放多少张卡片？

| 套餐 | 单次生成上限 | 文档总卡片上限 |
|------|------------|-------------|
| Free | 10 张 | 无限（手动添加） |
| Plus | 20 张 | 无限 |
| Pro | 20 张 | 无限 |
| Ultra | 20 张 | 75 张 |

单次 AI 生成最多 20 张，但你可以手动或用 Agent 追加更多。

### 7. 支持代码高亮吗？

支持。用 `/code` 插入代码块，支持常见语言的语法高亮（JavaScript、Python、TypeScript、Go、Rust 等）。在技术演示里放代码片段完全没问题。

### 8. 多人协作怎么用？

Free 版可以分享链接让别人查看。Team 及以上套餐支持实时多人编辑（类似 Google Docs）、评论、和基于角色的权限控制（查看 / 评论 / 编辑）。

### 9. 有移动端吗？

没有专门的 App，但 Gamma 的网页是响应式的——在手机浏览器里能正常查看和演示。编辑体验在手机上比较勉强，建议在电脑上编辑。

### 10. 什么场景应该用 Gamma，什么场景不该用？

```
用 Gamma 的信号：
✓ 需要快速出内容（< 30 分钟内要一份能看的演示）
✓ 内容为主、设计为辅（不需要像素级精确）
✓ 给人看的内容（demo、pitch、汇报、教程）
✓ 需要交互元素（嵌入视频、折叠区域）
✓ 需要 API 自动化批量生成

不用 Gamma 的信号：
✗ 需要精确到像素的设计（用 Figma）
✗ 需要完美的 PPTX 导出（用 PowerPoint + Copilot）
✗ 内容涉及公司高度机密且预算有限（用离线工具）
✗ 纯文字文档不需要视觉（用 Notion 或 Markdown）
✗ 需要复杂数据可视化（用 Tableau / Power BI 嵌入）
```

## 从 0 到 1 的推荐路线

1. **第一天**：注册免费账号 → 用 Generate 模式做一份试试手
2. **第一周**：试 Paste 模式和 Import 模式 → 掌握 Agent 编辑（`Cmd+E`）→ 用 Remix 改公开模板
3. **第二周**：设置品牌套件 → 建立你的常用模板 → 试试网站发布功能
4. **想更深**：升级 Pro → 用 API 做自动化 → 配 MCP 让 Claude 帮你调 Gamma

免费版足够让你判断 Gamma 适不适合你。如果你每周要做 2 份以上的演示内容，Pro 套餐的 $20/月基本第一次用就回本了。
