这一章手把手带你从注册 Gamma 账号到做出第一份能用的演示文稿。整个过程不超过 5 分钟，不需要任何设计基础。

![Gamma 创建界面 - 选择内容类型](https://images.gamma.app/b4ufuay1lzgjxjt/original/04c02e6ae2f24f42a9dbbb29b8dde8f3/gamma-create-view.png)

## 注册账号

1. 打开 [gamma.app](https://gamma.app)
2. 点 "Sign up free"，支持 Google 账号一键登录或邮箱注册
3. 注册后你会拿到 **400 个 AI 积分**（一次性的，用完不刷新）
4. 每次用 AI 生成内容大约消耗 40 积分，所以免费额度够做大约 10 份演示

注册完直接进工作台（Workspace），不用配置任何东西。

## 三种创建模式

Gamma 提供三种起步方式，看情况选：

| 模式 | 适合场景 | 怎么用 |
|------|---------|--------|
| **Generate（生成）** | 脑子里有想法，但没写好内容 | 输入一句话描述，AI 从零生成 |
| **Paste（粘贴）** | 已经有写好的文字、笔记 | 把文字粘进去，AI 帮你排版和补图 |
| **Import（导入）** | 手上有现成的 PDF / DOCX / PPTX | 上传文件，AI 转成 Gamma 卡片格式 |

90% 的情况用 Generate 就够了。

## 动手：做你的第一份演示

我们用 Generate 模式来做一个"React vs Vue 技术选型"的演示文稿。

**第一步：选内容类型**

进工作台后点 "Create new"，选 **Presentation**（演示文稿）。Gamma 还支持 Document（长文档）、Website（单页网站）和 Social Post（社交媒体帖子），但演示文稿是最常用的。

**第二步：写 Prompt**

在输入框里写：

```
对比 React 和 Vue.js 的技术选型，
面向有 1-2 年经验的前端开发者，
包含性能对比、生态系统、学习曲线、就业市场，
最后给出不同场景下的推荐，
用轻松专业的语气，不要太正式
```

Prompt 的关键是**说清楚受众和语气**。"给前端初级开发者看"和"给 CTO 决策用"产出的内容完全不同。

**第三步：审核大纲**

Gamma 会先给你一份大纲（Outline），通常 8-12 张卡片。你可以：
- 拖拽调整顺序
- 删掉不需要的卡片
- 手动加几个要点

别跳过这步——**大纲质量直接决定最终产出**。花 30 秒调一下大纲比后面逐页改省时间多了。

**第四步：选主题**

Gamma 提供几十个预设主题，也可以上传品牌色和 Logo。选一个接近你想要风格的就行，后面随时能换。

**第五步：生成**

点 "Generate"，等大约 60 秒。Gamma 会同时生成文字、配图、排版和设计元素。生成完直接就能用，但我建议你至少过一遍每张卡片，看看配图和措辞是否合适。

## 编辑和微调

生成后你可以直接在卡片上编辑：

- **点击文字**直接改内容
- **点击图片**可以换图（AI 重新生成或搜索 Unsplash / Pexels 免费图库）
- **拖拽卡片元素**调整布局
- **用 `/` 斜杠命令**插入新元素：

```
/image      → 插入图片
/chart      → 插入图表
/video      → 嵌入 YouTube / Vimeo
/toggle     → 插入折叠区域
/nested     → 插入嵌套子卡片
/code       → 插入代码块
/divider    → 插入分隔线
```

代码块支持语法高亮，所以在技术演示里放代码片段完全没问题。

## 分享和导出

做完后有几种分享方式：

1. **网页链接**（推荐）：点 "Share" 生成链接，对方在浏览器里打开就能看，保留所有交互效果
2. **PDF 导出**：格式比较忠实，但交互元素（折叠、嵌入视频）会丢失
3. **PPTX 导出**：能用，但复杂排版可能走样——如果一定要 PowerPoint 格式，建议生成时就选简单点的主题
4. **嵌入网页**：拿到 embed 代码贴到你的博客或文档站

## 免费积分策略

400 积分不多，别浪费在试探上。几个省积分的方法：

- 先用 Paste 模式（把已有内容粘进去排版），比 Generate 省积分
- 生成前把大纲调好，避免生成完再大改重新跑
- 小改动（换个词、调个颜色）不消耗积分，只有 AI 生成内容才扣
- 如果只是想看看效果，先看 Gamma 的公开模板库（gamma.app/templates），不花积分
