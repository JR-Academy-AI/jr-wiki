NotebookLM 上手门槛很低——有个 Google 账号就行，不用装任何软件，打开浏览器就能用。

![NotebookLM getting started](https://img.youtube.com/vi/px10ePAossA/maxresdefault.jpg)

## 注册和登录

1. 打开 [notebooklm.google.com](https://notebooklm.google.com)
2. 用你的 Google 账号登录
3. 完成——没有额外的注册步骤

如果你没有 Google 账号，先去 [accounts.google.com](https://accounts.google.com) 注册一个，然后回来登录。目前 NotebookLM 在大部分国家和地区都能用。

## 创建第一个笔记本

登录后你会看到一个干净的主页。点击 **"新建笔记本"（New Notebook）**，给它起个名字：

```
推荐的命名方式：
✅ "React 19 新特性研究"
✅ "毕业论文-文献综述"
✅ "公司 Q2 季度报告分析"

不推荐：
❌ "test"
❌ "新建笔记本 1"（以后找不到）
```

## 添加资料来源

笔记本创建后，最重要的一步：**喂资料**。NotebookLM 只能基于你上传的资料回答问题，不喂资料就是个空壳。

添加方式有几种：

| 方式 | 操作 | 适合场景 |
|------|------|---------|
| **上传文件** | 拖拽 PDF/DOCX/TXT 到页面 | 本地文档 |
| **粘贴链接** | 输入网页 URL | 在线文章、博客 |
| **YouTube** | 粘贴视频链接 | 教程视频（需有字幕） |
| **Google Drive** | 选择 Docs/Sheets/Slides | Google 生态用户 |
| **粘贴文本** | 直接复制粘贴内容 | 代码片段、笔记 |

**实操建议**：第一次用，选 2-3 篇你正在看的文档丢进去就行。别一上来就塞 50 个文件，先感受一下效果。

## 开始提问

资料加完后，左边是来源列表，右边是聊天区。直接用自然语言提问：

```
试试这些问法：

"这几篇文档的核心观点分别是什么？"
"文档 A 和文档 B 在 XX 问题上的观点有什么矛盾？"
"用通俗的语言给我解释一下文档里提到的 RAG 技术"
"帮我整理一份这些资料的摘要，按主题分类"
"基于这些 API 文档，写一个调用 /users 接口的示例"
```

每个回答都带数字引用标注，像这样：`NotebookLM 使用 RAG 技术来确保回答的准确性 [1][2]`。点击数字就跳到原文对应段落——这是 NotebookLM 最有价值的功能，让你随时能验证 AI 没有胡说。

## 生成 Audio Overview

这是 NotebookLM 的杀手级功能。点击右侧 **Studio 面板** → **Audio Overview** → **Generate**：

1. NotebookLM 会基于你的资料生成一段 5-20 分钟的播客
2. 两个 AI 主播用对话的方式讨论你的资料内容
3. 有说有笑、有举例有类比，完全不像机器人
4. 支持 80+ 种语言输出

生成之前可以点 **Customize** 自定义：

```
自定义 prompt 示例：

"用初学者能听懂的方式讲解，多用生活中的类比"
"重点讨论第三章的机器学习算法部分"
"用辩论的形式，让两位主播持不同观点"
"控制在 5 分钟以内，只讲最关键的要点"
```

生成的播客可以下载 MP3，也可以直接在页面上播放。我个人最喜欢的用法是：把论文丢进去生成播客，戴着耳机在通勤的时候听，比自己逐字读论文效率高不少。

## 使用 Studio 面板

除了 Audio Overview，Studio 面板还有更多工具：

- **Summary**：一键生成资料摘要
- **FAQ**：自动生成常见问题列表
- **Study Guide**：生成学习指南
- **Mind Map**：交互式思维导图，可以点击节点展开
- **Flashcards**：生成记忆卡片，适合考试复习

这些工具都在右侧 Studio 面板里，点一下就生成，不需要写任何 prompt。

## 分享和协作

做好的笔记本可以分享给别人：

1. 点击笔记本右上角的 **Share** 按钮
2. 输入对方的 Google 邮箱
3. 选择权限：**Viewer**（只读）或 **Editor**（可编辑）

协作者可以在同一个笔记本里提问、添加资料、生成笔记，适合团队一起做调研。
