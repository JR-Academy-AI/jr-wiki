## 为什么转行

做了 3 年土木工程师，年薪 $65k，天天跑工地。土木工程的系统思维在 DevOps 里特别有用 —— 基础设施就是基础设施，不管是物理的还是云上的。

## 学习路线

| 阶段 | 内容 | 时间 |
|------|------|------|
| Linux 基础 | 命令行、Shell 脚本 | 2 周 |
| AWS 核心 | EC2、S3、VPC、IAM | 4 周 |
| 容器化 | Docker、Docker Compose | 2 周 |
| CI/CD | GitHub Actions、Jenkins | 2 周 |
| Kubernetes | K8s 基础、Helm | 4 周 |
| IaC | Terraform | 3 周 |
| 项目实战 | 完整 CI/CD Pipeline | 3 周 |

## 转折点

第三个月考过了 AWS SAA，简历通过率从 10% 提到 40%。

## 建议

1. **先考 AWS SAA** —— 投入产出比最高
2. **所有东西都自己搭一遍** —— AWS Free Tier 够用
3. **土木的项目管理经验很值钱** —— DevOps 本质就是工程化管理
